# 상가 거리 봇

마스토돈 기반 텍스트 RPG 상점 봇입니다.
멘션 명령어로 아이템 구매/판매/장착, 음식 섭취, 타로 점술, 야바위 도박을 진행할 수 있습니다.

---

## 시작하기

### 요구 사항

- Node.js 18 이상
- 마스토돈 계정 및 액세스 토큰
- Google Cloud 서비스 계정 (Sheets 연동)
- Google Sheets 스프레드시트

### 설치
```bash
npm install
```

### 환경 변수 설정

프로젝트 루트에 `.env` 파일을 생성합니다.
```env
MASTODON_URL=https://your.instance.url
SHOP_TOKEN=your_mastodon_access_token

GOOGLE_SHEET_ID=your_google_sheet_id
GOOGLE_SERVICE_ACCOUNT_JSON={"type":"service_account","project_id":"..."}
```

### Google Sheets 설정
스프레드시트에 아래 탭과 헤더를 구성합니다.

**Players**
accountId | name | 지능 | 매력 | 체력 | 감성 | 사회성 | 도덕성 | 야망 | 위험도 | 의존성 | 스트레스 | 평판 | 전투 | 골드 | 턴 | 인벤토리 | 장착

**Items**
아이템명 | 상점 | 가격 | 판매비율 | 슬롯 | 최소나이 | 효과 | 설명

효과 컬럼은 JSON 문자열로 입력합니다.
{"체력":3,"스트레스":-2}

서비스 계정 이메일을 스프레드시트 편집자로 공유해야 합니다.

### 실행
```bash
node shop.js
```

## 명령어

모든 명령은 봇 계정을 멘션한 뒤 대괄호 `[ ]` 안에 입력합니다.

@상가봇 [명령어]
---

### 상점 목록 조회

| 명령 | 설명 |
|---|---|
| `[상가/무기상]` | 무기 · 방패 목록 |
| `[상가/의상실]` | 의상 목록 (12종) |
| `[상가/잡화점]` | 장신구 · 소비 아이템 목록 |
| `[레스토랑]` | 음식 메뉴 목록 (8종) |

---

### 구매 / 판매

| 명령 | 설명 |
|---|---|
| `[구매/아이템명]` | 아이템 구매. 음식은 구매 즉시 효과 적용, 인벤토리 미저장 |
| `[판매/아이템명]` | 아이템 판매. 장착 중인 아이템은 판매 불가 |

---

### 소비 아이템 사용

| 명령 | 설명 |
|---|---|
| `[사용/아이템명]` | 인벤토리의 소비 아이템 사용 후 제거 |

---

### 장착 관리

| 명령 | 설명 |
|---|---|
| `[장착/아이템명]` | 장착형 아이템 장착. 동일 슬롯 기존 아이템은 자동 교체 |
| `[제거/슬롯명]` | 해당 슬롯 아이템 해제 |

슬롯 종류: `무기` `방패` `의상` `장신구`

---

### 인벤토리 확인

| 명령 | 설명 |
|---|---|
| `[주머니]` | 소지금 · 인벤토리 · 장착 현황 출력 |

---

### 타로

| 명령 | 설명 |
|---|---|
| `[타로]` | 78장 중 3장을 뽑아 나이든 점술사의 지문으로 운세 출력 |

카드는 과거 / 현재 / 미래 3자리로 배치되며, 정방향(65%) · 역방향(35%) 랜덤 적용됩니다.

---

### 야바위

2단계 인터랙션 도박입니다. 판을 열면 컵 선택지가 출력되고, 선택 후 결과가 공개됩니다.

**1단계 — 판 개설**
[야바위/난이도/베팅액]
| 난이도 | 컵 수 | 성공률 | 성공 시 배율 | 설명 |
|---|---|---|---|---|
| 하 | 3개 | 약 33% | 2배 | 느린 딜러 |
| 중 | 3개 | 25% | 3배 | 속임수 있음 |
| 상 | 5개 | 약 17% | 5배 | 고수 딜러 |

최소 베팅: **10G** / 마이너스 골드 허용

**2단계 — 컵 선택**

하·중 난이도: [왼쪽]  [가운데]  [오른쪽]
상 난이도: [1번]  [2번]  [3번]  [4번]  [5번]

선택 후 5분이 지나면 판이 자동 취소됩니다.

## 아이템 슬롯 구분

| 슬롯 | 처리 방식 |
|---|---|
| `무기` `방패` `의상` `장신구` | 장착형 → `[장착]` / `[제거]` |
| 슬롯 없음 (잡화점) | 소비형 → 인벤토리 보관 후 `[사용/]` |
| 상점: 레스토랑 | 음식 → 구매 즉시 효과, 인벤토리 미저장 |
